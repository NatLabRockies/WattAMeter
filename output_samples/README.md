# WattAMeter Output Samples

This directory contains sample output files generated by the WattAMeter tool.

- The power measurements are stored in the `wattameter_<machine_id>.log`.
- The final results of the measurements are stored at `wattameter_emissions_<machine_id>.csv`.
- The log output of WattAMeter is stored in `wattameter-<job_id>-<machine_id>.txt`.
- The file `slurm-<job_id>-wattameter.txt` contains the output of the utility script used to run WattAMeter in a SLURM job.
